Installation
============

Dependencies
------------

Requirements:
    * Python 2.x (x>=6) or 3.x (x>=2)
    * PyQt4 4.x (x>=3 ; recommended x>=4) or PyQt5 5.x (x>=5)
    * spyderlib >=v2.0.10 for the test launcher
    * NumPy 1.x (x>=5)

Installation
------------

From the source package:

    `python setup.py install`
