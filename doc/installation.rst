Installation
============

Dependencies
------------

Requirements:
    * Python 2.x (x>=6)
    * PyQt4 4.x (x>=4) or PyQt5 5.x (x>=5)
    * NumPy 1.x (x>=5)

Installation
------------

From the source package:

    `python setup.py install`

----------------
