Installation
============

Dependencies
------------

Requirements:
    * Python 3.x (x>=4)
    * PyQt4 4.x (x>=4), PyQt5 5.x (x>=5), PyQt6 or PySide6
    * QtPy >= 1.3
    * NumPy 1.x (x>=5)
    * Sphinx 1.x (x>=1) for documentation generation

Installation
------------

From the source package:

    `python setup.py install`
